Visando a necessidade de otimizar o gerenciamento de produtos e estoque em lojas de pequeno e médio porte, que frequentemente utilizam métodos manuais sujeitos a erros e ineficiência, desenvolveu-se o Estoque Fácil, um sistema web de controle de estoque. O projeto foi auxiliado pela linguagem de programação Java e a plataforma de desenvolvimento Play Framework 1.5.3, utilizando o padrão de arquitetura Model-View-Controller (MVC) e o Banco de Dados H2 para persistência dos dados. Assim, espera-se que, por meio deste trabalho, seja simplificada a rotina de controle de estoque e viabilizada a redução de perdas e erros operacionais, provendo aos gestores uma ferramenta eficiente para o Cadastro, Listagem, Edição e Exclusão (CRUD) de produtos, contribuindo para uma tomada de decisão mais ágil e informada.
